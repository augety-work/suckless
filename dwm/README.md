## Shortcut key (快捷键)
| 快捷键                 | 结果            |
|------------------------| --------------- |
| `Win + p`              | 打开菜单        |
| `Win + Return`         | 打开终端        |
| `Win + r`              | 刷新壁纸        |
| `Win + a`              | 截屏            |
| `Win + '`              | 打开浮动终端    |
| `Win + Shift + Return` | 设置主窗口      |
| `Win + b`              | 关闭/打开状态栏 |
| `Win + Tab`            | 切换标签页      |
| `Win + Shfit + q`      | 关闭窗口        |
| `Win + Shift + c`      | 关闭 DWM        |
